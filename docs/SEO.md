## 5. Технический SEO-чеклист (для Claude Code)

```
[ ] Canonical на каждой странице
[ ] Sitemap.xml генерируется автоматически (Laravel route model)
[ ] robots.txt: закрыть /admin, /api, открыть всё публичное
[ ] Alt-тексты на всех изображениях — обязательно для работ студентов
[ ] Страница 404 — с навигацией и кнопкой на главную
[ ] HTTPS + редирект с http://
[ ] Хлебные крошки на страницах специальностей (BreadcrumbList schema)
[ ] OG-теги на всех страницах (title, description, image)
[ ] Мобильная версия — PageSpeed ≥ 75 на мобайл
[ ] LCP < 2.5s (особенно hero с фото работ студентов — оптимизация изображений)
[ ] Schema.org валидна: schema.org/validator
[ ] Метрика: счётчик из SiteSettings::get('metrika_id')
[ ] Цели Метрики: lead_submit / dod_signup / call_click / shop_purchase
```
