## 1. Текущий сайт клиента — collegefaberge.mskobr.ru

**Платформа:** mskobr.ru — стандартный шаблон Департамента образования Москвы. Никакого собственного дизайна.

| Критерий | Оценка |
|----------|--------|
| Конверсионная механика | ❌ Нет понятного CTA, нет форм на главной |
| Структура и иерархия | ❌ Информационный портал, не промо. Логика: «всё обо всём» |
| Контент | ❌ Официальный язык, канцелярит, нет вдохновения |
| Мобильная версия | ⚠️ Работает технически, но не адаптирована под абитуриента |
| SEO-сигналы | ⚠️ Title есть, H1 стандартный шаблонный, schema нет |

**Вывод:** Сайт решает задачу «документальное присутствие», не задачу «привлечение абитуриентов». Промо-сайт строим с нуля — брать с официального нечего, кроме фактических данных (специальности, адреса, контакты).

## 2. Конкурентный анализ

### IThub (ithub.ru) — ключевой референс по механике

**Сильное:**
- Слоган «Готовим лидеров цифрового мира» — продаёт будущее, не специальность
- ДОД как главный CTA: отдельная страница, форма записи, анонсы мероприятий ежемесячно
- Карточки специальностей с фильтрацией
- Истории выпускников с реальными местами работы
- Живой тон — молодёжный, без канцелярита

**Слабое:**
- Перегружен информацией на главной
- Визуально устарел — много текста, мало воздуха
- Нет strong visual identity — сайт мог бы быть про что угодно

**Что берём для Фаберже:**
- Механику ДОД как главного конверсионного события
- Истории выпускников с конкретными местами работы
- Тон: говорим про будущее, не про учёбу

### abit.itmo.ru — эталон абитуриентского раздела

**Сильное:**
- Отдельный домен/раздел только для абитуриентов — фокус
- Фильтрация программ по направлениям (поиск + кнопки-теги)
- Личный кабинет абитуриента — даты, статус заявки
- ДОД анонсируются прямо на главной с датой и кнопкой регистрации
- Слоган «Всё складывается в ИТМО» — эмоциональный, не информационный

**Что берём:**
- Структуру карточки специальности (название → описание → что изучают → куда идут работать → поступить)
- ДОД — анонс с датой прямо на главной, не в новостях
- Фильтрация специальностей — кнопки-теги по направлениям

### 26kadr.ru — образовательный портал

**Сильное:**
- Каталог специальностей с описаниями
- Структурированный поиск

**Слабое:**
- Агрегатор, не брендированный сайт — холодный тон
- Нет эмоции, нет истории

**Что берём:** только логику каталога специальностей

### samgik.ru (работа SWIPE) — прямой аналог

**Сильное:**
- Сочетание академичности и современности
- Реальные фото студентов и пространств
- Чёткая иерархия разделов

**Что берём:** визуальный тон — серьёзный институт с живым характером

## 3. Ключевые инсайты

### Инсайт 1 — главный
Все сильные образовательные сайты продают **не специальность, а личность которой ты станешь**. IThub: «лидер цифрового мира». ИТМО: «всё складывается». Для Фаберже формула: **«Ты создашь то, что останется»** — ювелир, дизайнер, художник создаёт вещи которые переживут их авторов.

### Инсайт 2 — что никто не делает хорошо
Никто из конкурентов не показывает **реальные работы студентов** как главный аргумент. У Фаберже есть уникальное преимущество — изделия которые можно потрогать и купить. Это сильнее любого описания специальности.

### Инсайт 3 — ДОД как воронка
ДОД — самый низкобарьерный первый шаг. Не «подать документы», не «поступить» — просто «прийти и посмотреть». IThub строит вокруг этого всю конверсию. Для Фаберже это особенно важно — творческие специальности надо пощупать, увидеть мастерские, почувствовать атмосферу.

### Инсайт 4 — визуальная ниша
Все образовательные сайты визуально похожи: синие/зелёные корпоративные цвета, стоковые фото студентов за ноутбуками. Фаберже — колледж декоративно-прикладного искусства. Сайт может и должен выглядеть как культурное учреждение: тёплые тона, фактура материалов, художественная типографика. Это само по себе отстройка.

## 4. Контент-инвентарь

| Тип контента | Статус | Что делать |
|-------------|--------|-----------|
| Фотографии | Стоки на старте | Нужна профессиональная съёмка: мастерские, работы студентов, педагоги |
| Логотип | Есть на официальном сайте | Запросить векторный SVG у клиента |
| Тексты специальностей | Нет | Писать с нуля по шаблону |
| Цифры (99%, 70%, 150к) | Заглушки | Верифицировать с клиентом перед запуском |
| Истории выпускников | Нет | Собирать через клиента (формат: имя, специальность, где работает сейчас) |
| Список баз практики | Частично (бриф) | Уточнить полный список у клиента |
| Контакты и адреса | Заглушки | 3 адреса корпусов — верифицировать |
| Документы для поступления | Нет | Получить от клиента (PDF) |
| Даты ДОД | Нет | Получить от приёмной комиссии |

## 5. Инсайты для агентов

**→ IA:** Строить структуру главной вокруг визуального удара — сначала показываем работы студентов, потом объясняем где этому учат. Блок ДОД — не в футере, а в середине главной как отдельный конверсионный момент.

**→ Content:** Тон — кураторский, как музейный гид который влюблён в своё дело. Не «у нас 9 специальностей», а «выбери свой путь в искусстве». Цифры 99% / 70% / 150к — только после верификации, пока ставим тематические факты про выпускников (Большой театр, Жостовская фабрика).

**→ SEO:** Основные запросы — «колледж дизайна Москва», «поступить на ювелира Москва», «учиться на флориста Москва», «декоративно-прикладное искусство колледж». Каждая специальность — отдельная SEO-страница с геозапросом.

**→ Design:** Уйти от «корпоративного образования». Взять как ориентир визуальный язык культурных институций — Третьяковка, ГЦСИ. Тёплая палитра, serif-типографика, фактура. Фотографии рук за работой важнее фотографий аудиторий.

## 6. Итоговый вывод

Рынок образовательных сайтов в Москве выглядит одинаково — корпоративно и скучно. Фаберже имеет уникальную возможность занять нишу «образовательный сайт как культурное медиа» — показывать творческий процесс, реальные работы, живых людей. Главный конверсионный путь первой очереди: вдохновить → показать специальность → привести на ДОД.
