> **UPDATE 2026-04-22 (PM-решение):** страницы специальностей консолидированы.
> Один канонический URL на специальность — `/specialnosti/{slug}`. Маршрут

### 01 · Hero
**Что посетитель думает:** «Что это такое? Это для меня?»
**Задача блока:** За 3 секунды передать характер — это не обычный колледж, здесь создают настоящее искусство
**Содержимое:**
- Фоновое изображение / видео: крупный план работы студента (руки за работой — ювелирка, роспись, текстиль)
- H1: «Учим тех, кто создаёт шедевры»
- Подзаголовок: вдохновляющая строка про профессию и будущее
- Два CTA рядом:
  - Primary: «Поступить» → source: hero → форма
  - Secondary: «Смотреть работы» → anchor к блоку 03
- ДОД-баннер (временный, управляется из Orchid):
  - «День открытых дверей — [дата]» + кнопка «Записаться» → source: dod-banner → /abiturientam

**Чего нет:** цены, перечисление специальностей, казённый язык

## Структура раздела «Абитуриентам» /abiturientam

### Шапка раздела
- Hero: «Кем хочешь стать?» / «Сделай имя в искусстве»
- Два типа кнопок-фильтров: ПРОФЕССИЯ / СПЕЦИАЛЬНОСТЬ
- Поисковая строка (код или название направления)
- Кнопки-направления: Дизайн / Декоративно-прикладное / Ювелирное / Флористика / Текстиль / Графика

### Карточки специальностей (9 шт.)
На хабе — **только краткое превью**: фото, название, тип (специальность/профессия),
одна строка УТП, кнопка «Узнать больше» → `/specialnosti/[slug]`.
Полное содержание специальности (все 12 блоков) живёт на странице специальности
(см. ниже «Структура шаблона специальности»), не дублируется на хабе.

### Дополнительные блоки раздела
- **Календарь абитуриента:** даты ДОД / начало подачи документов / сроки зачисления
- **Какие документы нужны:** список с возможностью загрузки PDF
- **Подготовительные курсы:** форма записи
- **Приёмная комиссия (дубль):** адрес + телефон + email + карта
- **Где находимся (дубль):** 3 корпуса

## Структура шаблона специальности /specialnosti/[slug]

1. Чему научишься (УТП + задачи + материалы)
2. Какой получишь диплом (квалификация)
3. Кем можешь работать (базы работодателей)
4. Сколько будешь получать (вилка зарплат)
5. Карьерный рост (от и до)
6. Как поступить (база классов + испытания + баллы)
7. Про учёбу (форма / бюджет / срок / онлайн)
8. Где практиковаться (базы практики + мастерские)
9. Какие предметы (перечень)
10. Фото учебного процесса и работ
11. Адрес корпуса
12. CTA: «Поступить» → форма + «Подготовительные курсы» → форма
